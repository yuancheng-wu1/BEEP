# BEEP
Build Elegant and Efficient Plots(BEEP)

BEEP is a shiny app built on ggplot2. It provides an interactive interface to visualize data in research. It is developed to make plotting efficient, saving time for replicative work when presenting various plots in R.

Declaration: 
This app is a collaborated work with Chatgpt, and is still under construction. More functions are on the way.
