# BEEP
Build Elegant and Efficient Plots(BEEP)

BEEP is a shiny app built on ggplot2. It provides an interactive interface to easily visualize data in research. It is developed to make plotting efficient, saving time for replicative work when presenting various plots in R.

Declaration: 
This app is a collaborated work with Chatgpt, and is still under construction. Basic plots are currently available. More functions are on the way.
